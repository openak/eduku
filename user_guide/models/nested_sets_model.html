Nested Sets Model

The nested sets model class was created by 'thunder uk' on the CodeIgniter forums, there is a inactive thread here. I did not create this class but BackendPro uses it in the administration control panel to allow the user to manage access control permissions. For full documentation on its methods please see system/application/models/Nested_sets_model.php.
